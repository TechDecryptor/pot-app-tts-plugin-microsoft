## 使用方法

### 安装 Pot

此插件为 [Pot App](https://github.com/pot-app/pot-desktop) 翻译插件，必须配合 [Pot App](https://github.com/pot-app/pot-desktop) 来使用

### 安装插件

1. 下载对应平台的插件，解压得到 `.potext` 文件
2. 打开 Pot-偏好设置-服务设置-翻译-添加外部插件-安装外部插件
3. 选择刚刚解压得到的 `.potext` 文件，安装成功
4. 将插件添加到服务列表即可使用
